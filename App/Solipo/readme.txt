[introduction]
Solipo is GUI wrapper application for polipo on Windows.
Solipo is FREE software.

solipoXXXX.zip contains original polipo source code in polipo folder.

[how to use solipo]
1. put the folder "solipo" into a certain place ("Program Files" folder
   is recommended).
2. run "solipo.exe".
3. set proxy address to "127.0.0.1" and port to "8123" on your browser.

"forbidden" and "uncachable" are loaded by polipo as default.  If you 
would like to ignore those settings, please rename those files and 
restart polipo.

[はじめに]
solipo は、Windows 上で動作する polipo の GUI ラッパーアプリケーション
です。solipo はフリーウェアです。

オリジナル polipo のソースコードは polipo フォルダにあります。

[使い方]
1. 解凍した "solipo" フォルダを適当な場所に置きます。
2. 「solipo.exe」を起動します。
3. http / https (ssl) のプロキシアドレス「127.0.0.1」・ポート「8123」に
   設定します。

デフォルトで、forbidden / uncachable の設定が有効になっています。これ
らの機能を利用しない場合、ファイル名を変更して restart するか、config
ファイルの設定を変更してください。
